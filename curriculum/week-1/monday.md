**Community Check-In: "Welcome Circle"**

   - Walk students through the physical space: tables, supply area, cozy reading corner, math manipulative shelf, portfolio folders.
   - Introduce the **Visual Daily Schedule** posted on the wall. Explain each block using picture + word cards.
   - Show the **Self-Monitoring Checklist** (laminated, one per student). Walk through each item:
     - Did I listen when someone was talking?
     - Did I try my best?
     - Did I ask for help when I needed it?
     - Did I follow directions the first time?
   - Students write their names on their checklists with dry-erase marker.

3. **SEL Connection: "Feelings Check-In" (5 min)**:
   - Post a feelings chart with 6 faces (excited, happy, nervous, calm, tired, unsure).
   - Each student points to or names how they feel arriving today. Teacher validates all feelings: *"It makes sense to feel nervous on a first day. By Friday, this will feel like home."*
   - Establish the routine: "Every morning we'll check in — there are no wrong feelings here."

4. **Brain Game: "Stand Up If…" (2 min)**:
   - "Stand up if you rode in a car to get here today."
   - "Stand up if you have a pet."
   - "Stand up if you like math." / "Stand up if math is hard sometimes."
   - "Stand up if you're ready to have an awesome summer."
   - Purpose: movement, connection, normalizing that learning can be hard.

### Math Block (9:20–10:20)

**Objective**: Students will represent numbers to 10,000 (3rd) / 100,000 (4th) using base-ten blocks and place value charts, identifying the value of each digit. (IEP Goal: Number Sense — place value understanding.)

**Materials**:
- Base-ten blocks (units, rods, flats, cubes) — 1 set per pair of students
- Place value discs — 1 set per student
- Desktop number lines (0–10,000)
- Dry-erase boards + markers (1 per student)
- Place Value Diagnostic worksheet (pre-printed, 1 per student)
- Chart paper + marker (for anchor chart)
- Place value chart template (laminated, 1 per student)

#### Warm-Up/Activation (5 min)
- **"Number of the Day"**: Write the number **2,547** on the board.
- Ask: *"What do you notice about this number? Turn to the person next to you and share one thing."*
- Collect 3–4 responses. Accept all answers (it's even, it has a comma, there's a 2 in the thousands, etc.).
- Say: *"Today we're going to explore what each digit in a number is worth — its place value."*

#### Direct Instruction (15 min) — CRA Level: Concrete → Representational
1. **Concrete** (8 min): Hold up a unit cube, a rod, a flat, and a large cube.
   - "This unit is worth 1. How many units make a rod?" (10) Have students count to verify.
   - "How many rods make a flat?" (10 — that's 100) Have students count edges.
   - "How many flats make a cube?" (10 — that's 1,000)
   - Build the number **2,547** together using blocks at their tables:
     - 2 cubes (thousands), 5 flats (hundreds), 4 rods (tens), 7 units (ones)
   - Students build along with you. Walk the room to observe.

2. **Representational** (7 min): Draw a place value chart on chart paper.
   - Label columns: Thousands | Hundreds | Tens | Ones
   - Record 2,547 in the chart. Draw quick-pics (dot for thousands, square for hundreds, line for tens, dot for ones).
   - Show how the same number looks as blocks, in a chart, and in standard form.
   - Ask: *"What is the value of the 5 in 2,547?"* (500, not 5)
   - Key question: *"What's the difference between the digit and its value?"*

#### Guided Practice (15 min)
- **"Build It, Draw It, Write It" Activity**: Teacher calls out a number. Students:
  1. Build it with base-ten blocks (concrete)
  2. Sketch it using quick-pics on dry-erase boards (representational)
  3. Write it in standard form and expanded form (abstract)

- Numbers to use (progressive difficulty):
  - 346 (warm-up, all students)
  - 1,208 (zero in the tens — key misconception check)
  - 4,063 (zero in the hundreds)
  - 7,891 (all non-zero digits)
  - **For 4th graders who finish quickly**: 23,456 and 50,102

- After each number, ask one student to explain their thinking: *"How did you know where to put the zero?"*

#### Independent Practice (10 min)
- **Place Value Diagnostic Worksheet** (this is the assessment for today):
  - Section A: Write the value of the underlined digit (6 problems)
  - Section B: Write the number in expanded form (4 problems)
  - Section C: Compare using >, <, = (4 problems)
  - Section D: Round to the nearest 10 and 100 (4 problems)
  - Section E (4th grade extension): Place value to 100,000 (4 problems)

- Students work independently. Teacher circulates, noting:
  - Who counts on fingers vs. knows place value automatically?
  - Who struggles with zeros in the middle of numbers?
  - Who confuses the digit with its value (e.g., says the 5 in 2,547 is worth 5)?

#### Closure (5 min)
- Collect diagnostic worksheets.
- **Exit Ticket** (verbal, thumbs up/down): *"The 3 in the number 4,312 is worth 300. Thumbs up if you agree, thumbs down if you disagree."* (Answer: agree)
- *"The 0 in 3,074 means there are no ones."* (Answer: disagree — 0 is in the hundreds place)
- Preview tomorrow: *"Tomorrow we'll use number lines to place and compare numbers."*

#### Data Collection
- Score diagnostic worksheet tonight (record per-student: /6 digit value, /4 expanded form, /4 comparison, /4 rounding)
- Note observational data during guided practice: who needed physical blocks vs. who moved to drawing quickly
- Flag students who struggled with zero as a placeholder — reteach target for Tuesday

#### Differentiation
- **Below level**: Provide a pre-filled place value chart with column headers. Limit numbers to 3 digits (hundreds). Use only base-ten blocks (stay concrete).
- **On level**: Work with 4-digit numbers as described above. Move between concrete and representational.
- **Above level**: Extend to 5- and 6-digit numbers. Challenge: "Can you write a number where the digit 4 has a value of 4,000? What about 40,000?"

#### Part 1: OG Lesson Introduction — Whole Group (10:30–11:00)

**Objective**: Students will identify and produce sounds for all 26 consonant and 5 short vowel phonogram cards with ≥70% accuracy during initial review. (IEP Goal: Decoding — letter-sound correspondence.)

**Materials**:
- OG phonogram cards: 26 consonants + 5 short vowels (a, e, i, o, u)
- Sand/salt trays (1 per student)
- Dry-erase boards + markers
- "My Summer Learning Journal" composition notebooks (1 per student)
- Phonics screener forms (for individual pulls)
- Timer

**OG Lesson Structure**:

1. **Review — Phonogram Drill (8 min)**:
   - Hold up phonogram cards one at a time: consonants first, then short vowels.
   - Students respond chorally with the sound (not the letter name): /b/, /k/, /d/…
   - For short vowels, use the key word association:
     - a = apple /ă/
     - e = elephant /ĕ/
     - i = itch /ĭ/
     - o = octopus /ŏ/
     - u = up /ŭ/
   - Go through the deck twice. First time: teacher models, students echo. Second time: students respond independently.
   - Note which sounds students hesitate on or confuse (common: /b/ and /d/, short /e/ and short /i/).

2. **New Concept — Introduction to OG Routine (7 min)**:
   - Explain: *"Every day in our reading time, we'll follow the same steps. This is called structured literacy. It helps your brain learn to read in a really organized way."*
   - Post the OG Lesson Steps anchor chart:
     1. Review (phonogram cards — what we just did)
     2. New Sound or Pattern (learn something new)
     3. Practice Reading Words (use our new pattern)
     4. Dictation (teacher says it, you write it)
     5. Connected Text (read a real passage)
   - *"Today is a review day, so we'll practice steps 1, 3, and 4."*

3. **Guided Practice — CVC Word Reading (8 min)**:
   - Write on the board: **sat, pin, hug, bed, hot**
   - Model the first word using finger tapping: hold up 3 fingers, tap each finger as you say each sound, then sweep fingers together to blend: /s/ - /ă/ - /t/ → "sat"
   - Students practice finger tapping with the remaining 4 words.
   - Then try: **flat, drum, ship, chest, stump** (blends and digraphs — observe who can do these)
   - Students read each word. If they struggle, guide them to tap through each sound.

4. **Dictation Practice (7 min)**:
   - Distribute sand/salt trays.
   - *"I'm going to say a word. First, you'll repeat it. Then tap it. Then write it in the sand."*
   - Dictation words: **sit, map, bug, hen, log**
   - After each word, students hold up their trays to show. Teacher gives a quick thumbs-up or corrects.
   - For the last 2 minutes, students transfer their best 3 words to dry-erase boards (practicing pencil-to-paper transfer).

**Data Collection**:
- During the phonogram drill, tally on a class roster: mark any phonogram card a student missed or hesitated on (>2 sec delay = mark it)
- During CVC reading, note who could decode blends/digraphs vs. who could not
- Sand tray dictation: note letter reversals (b/d, p/q) and vowel confusions

**Phonics Screener — Individual Pulls (3–4 min per student)**:
- Use the Blevins Quick Phonics Screener (or equivalent).
- Part 1 covers: letter names, letter sounds, CVC words, short vowels in isolation.
- Pull students in alphabetical order (aim to get through 4–5 students today; continue Tuesday).
- Record results on the screener form. Star any patterns that are not yet mastered.

**Journal Setup — Independent Work (remaining students)**:
- Each student receives a composition notebook.
- Instructions posted on chart paper:
  1. Write your name on the front cover.
  2. On the first page, write: **"My Summer Learning Journal"** and today's date.
  3. Decorate the cover with drawings of things you like (5 min).
  4. On the second page, write the heading: **"About Me as a Learner"**
  5. Copy and complete these sentence starters (posted on board):
     - "Reading is ___ for me because ___."
     - "In math, I am good at ___."
     - "One thing I want to get better at this summer is ___."
     - "Something people might not know about me is ___."
  6. If you finish, draw a picture of yourself on the next page.

- A teaching assistant or volunteer monitors the journal group. If no assistant is available, set a visual timer for 5-minute intervals and circulate between screener pulls and the journal group.

**Data Collection**:
- Phonics screener scores recorded on individual student forms
- Collect journals at the end of the day — read the "About Me" responses for insight into student self-perception and attitudes toward learning
- Note who completed all journal prompts independently vs. who needed prompting/support

**Differentiation**:
- **Below level**: Provide sentence starters on a handout they can glue into their journal (reduces the copying burden). Allow drawing as an alternative to writing for the "About Me" page.
- **On level**: Complete the journal prompts as written.
- **Above level**: After completing prompts, write a full paragraph about themselves as learners (no sentence starters — free write).

### Exploration & Enrichment (12:00–1:00)

**Activity**: Nature Walk & Observation Journal

**Enrichment Category**: Nature & Parks (Monday theme)

**Overview**: Students take a guided walk around the outdoor space near the program facility. They practice observation skills, collect natural items, and record what they see in their nature journals. This builds descriptive vocabulary (reading/writing connection) and counting/sorting skills (math connection).

**Materials**:
- Nature journals (blank sketch + lined pages, 1 per student)
- Pencils + colored pencils (carried in a ziplock bag per student)
- Magnifying glasses (1 per student or 1 per pair)
- Paper bags for collecting natural objects (1 per student)
- Clipboard (1 per student, to write on while outside)
- Chart paper for group debrief (back inside)

**Schedule**:

1. **Introduction (5 min, inside)**:
   - *"Scientists and writers both start by observing — looking really carefully at the world around them. Today we're going outside to practice being observers."*
   - Rules for outdoor time: stay with the group, don't pick living plants (only fallen items), respect the space.
   - Model an observation: hold up a leaf. *"I notice it has a stem, it's green with some brown edges, it has lines running through it — those are called veins. It feels smooth on top and a little rough on the bottom. It's about the size of my palm."*
   - *"You're going to observe like that — use your eyes, your hands, and your words."*

2. **Nature Walk (20 min, outside)**:
   - Walk a planned route around the outdoor area (parking lot perimeter, nearby park, sidewalk garden beds, etc.).
   - At 3–4 stopping points, pause and prompt:
     - Stop 1: *"Look down at the ground. What do you see? Find 3 different things and sketch one in your journal."*
     - Stop 2: *"Look up. What do you notice about the sky, the trees, the buildings? Write 2 describing words."*
     - Stop 3: *"Use your magnifying glass. Find something tiny and draw it BIG in your journal."*
     - Stop 4: *"Collect 3–5 natural objects in your bag — leaves, pebbles, sticks, seeds. We'll use these inside."*
   - Teacher circulates, asks prompting questions: *"What does it feel like? What does it remind you of? How many do you see?"*

3. **Sorting & Counting Activity (15 min, inside)**:
   - Back at tables, students empty their bags and sort their natural objects.
   - Math connection: *"Sort your objects into groups. How did you sort them? Can you sort them a different way?"*
   - On dry-erase boards, record: how many in each group, which group has the most/least.
   - Reading/writing connection: *"Choose your favorite object. In your nature journal, write 3 sentences describing it. Use at least 2 describing words (adjectives)."*
   - Share: 2–3 students read their descriptions aloud. Class tries to guess which object it describes.

4. **Wrap-Up & Portfolio (10 min)**:
   - Students tape or glue a small leaf or flat item onto their nature journal page (if appropriate).
   - Teacher takes a photo of each student with their collected objects (for portfolio / parent communication).
   - Clean up natural objects (some may be stored for a Wednesday STEM activity).
   - **Portfolio piece**: The nature journal page (sketch + 3 sentences) becomes the first entry in their enrichment portfolio.

5. **Closing Circle (10 min)**:
   - Return to morning circle formation.
   - Each student shares one word about their day (go around the circle quickly).
   - Introduce end-of-day routine: pack up folder, check self-monitoring checklist, put chair in.
   - Teacher reads each student's name and gives a personal positive: *"[Name], I noticed you helped [other student] find a magnifying glass. That's what a learning community does."*
   - Preview tomorrow: *"Tomorrow you'll bring an item from home that tells us something about you. Think about what you want to share."*
   - Dismissal at 1:00.

**Academic Connection**:
- **Reading/Writing**: Descriptive vocabulary, observation as a pre-writing strategy, using adjectives in sentences.
- **Math**: Sorting and classifying objects, counting, comparing quantities (more/fewer/equal).
