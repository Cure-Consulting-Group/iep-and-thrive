# Week 1 — "Foundations & Assessment"
## Theme: Who We Are as Learners

**Dates**: July 7–11, 2026
**Focus**: Establish routines, build community, assess baselines, determine OG entry points

---

## Weekly Learning Objectives

### Reading & Phonics (OG)
- Review closed syllable patterns (CVC, CCVC, CVCC) with multisensory drill
- Assess consonant blend and digraph mastery (sh, ch, th, wh, ck)
- Introduce/review OG lesson structure (review → new → practice → dictation → connected text)
- Establish individual OG entry points via phonics screener
- Begin red word instruction (Tier 1: the, was, is, are, have, come, some, one)

### Comprehension
- Introduce active reading strategies: predict, monitor, retell
- Practice retelling using beginning/middle/end framework
- Set up reading response journals

### Writing
- Collect baseline writing sample (personal narrative prompt)
- Review sentence structure: subject + predicate
- Set up "My Summer Learning Journal"
- Practice sentence writing with correct capitalization and end punctuation

### Math
- Assess place value understanding (to 10,000 for 3rd / 100,000 for 4th)
- Review rounding to nearest 10, 100, 1,000
- Practice number line placement and comparison (>, <, =)
- Assess addition/subtraction fact fluency (diagnostic)

### Executive Function & SEL
- Establish classroom routines and expectations
- Introduce visual daily schedule
- Practice self-monitoring checklist ("Did I…?")
- Community building: "About Me" sharing, team-building games
- Goal-setting: "What I want to learn this summer"

## Differentiation Notes

### Flexible Grouping (Determined by Thursday)
Based on pre-assessment data, students will be placed in flexible groups:

**Phonics Groups** (for OG instruction):
- Group A: Still solidifying CVC, blends → Start at OG Level 1 review
- Group B: Blends/digraphs solid, vowel teams emerging → Start at OG Level 2
- Group C: Vowel teams partially mastered → Start at OG Level 3

**Math Groups**:
- Group A: Place value to 1,000, working on addition/subtraction fluency
- Group B: Place value to 10,000, addition/subtraction fluent, ready for multiplication concepts
- Group C: Place value solid, some multiplication exposure, ready for deeper work

**Note**: Groups are flexible and will be adjusted every 2 weeks based on progress monitoring data.
