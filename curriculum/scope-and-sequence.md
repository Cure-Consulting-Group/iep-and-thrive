# IEP & Thrive — Summer 2026 Scope & Sequence
## 6-Week Curriculum Overview | Grades 3–4 | Orton-Gillingham Framework

**Designed by**: Curriculum Specialist, IEP & Thrive
**Framework**: Orton-Gillingham (Structured Literacy) + CRA Math
**Standards Alignment**: NYS CCLS ELA & Math, Grades 3–4
**IEP Goal Domains**: Reading Fluency, Decoding, Comprehension, Written Expression, Math Computation, Math Problem-Solving, Executive Function

---

## Program Parameters

| Parameter | Detail |
|-----------|--------|
| Duration | 6 Weeks (July 7 – August 15, 2026) |
| Schedule | Monday – Friday, 9:00 AM – 1:00 PM |

| Grades | Rising 3rd–4th |

## 6-Week Overview Grid

### WEEK 1 — "Foundations & Assessment"
*Theme: Who We Are as Learners*

| Block | Focus | Details |
|-------|-------|---------|
| **Reading/Phonics** | Review & Diagnostic | Short vowels (CVC), consonant blends, digraphs (sh, ch, th, wh, ck); Closed syllable review; High-frequency red words assessment; OG entry point established per student |
| **Comprehension** | Active Reading Strategies | Making predictions, monitoring comprehension, retelling (beginning/middle/end); Introduction of reading response journals |
| **Writing** | Baseline Writing Sample + Journal Setup | Personal narrative prompt (diagnostic); Set up "My Summer Learning Journal"; Sentence structure review (subject/predicate) |
| **Math** | Number Sense Foundation | Place value to 10,000 (3rd) / 100,000 (4th); Rounding and estimation; Number line placement and comparison; Addition/subtraction fact fluency diagnostic |
| **Enrichment Theme** | "Getting to Know Our World" | Mon: Nature walk + observation journal; Tue: "All About Me" museum of personal artifacts; Wed: Build a structure that represents you (STEM); Thu: Classroom economy introduction (life skills); Fri: Self-portrait + goal-setting art |
| **Assessment** | Pre-Assessment Battery | ORF (oral reading fluency), phonics screener, sight word inventory, math computation probe, writing sample |
| **IEP Goals Addressed** | Decoding, fluency, number sense, written expression |

---

### WEEK 2 — "Patterns & Connections"
*Theme: Patterns in Language and Numbers*

| Block | Focus | Details |
|-------|-------|---------|
| **Reading/Phonics** | Vowel Teams (Part 1) | Vowel teams: ai/ay, ee/ea, oa/ow; Multisensory drill: sky writing, sand trays, phonogram cards; Decodable text practice with target patterns; Red words: said, they, were, could, would |
| **Comprehension** | Main Idea & Supporting Details | Identifying topic vs. main idea; Finding supporting details in informational text; Graphic organizer: main idea web |
| **Writing** | Paragraph Structure | Topic sentence, detail sentences, closing sentence; Modeled → shared → guided → independent writing progression; Paragraph about a pattern observed in nature (enrichment connection) |
| **Math** | Multiplication Concepts (Part 1) | Equal groups with manipulatives (concrete); Arrays: building and drawing; Skip counting connection to multiplication; Fact strategies: x0, x1, x2, x5, x10 |
| **Enrichment Theme** | "Patterns Everywhere" | Mon: Nature patterns — symmetry in leaves, spirals in shells; Tue: Patterns in art & architecture (museum/virtual); Wed: Pattern-based coding challenge (STEM); Thu: Recipe patterns — doubling/halving ingredients; Fri: Tessellation art + pattern poetry |
| **Assessment** | Weekly Probe #1 | ORF passage, phonics pattern sort (vowel teams), multiplication facts timed probe |
| **IEP Goals Addressed** | Decoding (vowel teams), comprehension (main idea), math computation (multiplication) |

---

### WEEK 3 — "Building Blocks"
*Theme: How Things Are Built — Words, Numbers, Structures*

| Block | Focus | Details |
|-------|-------|---------|
| **Reading/Phonics** | Vowel Teams (Part 2) + R-Controlled Vowels | Vowel teams: oi/oy, ou/ow, au/aw; R-controlled vowels: ar, or, er/ir/ur; Syllable type review: closed, open, VCe, vowel team, r-controlled; Multisensory: arm tapping for syllable division |
| **Comprehension** | Story Elements & Sequencing | Characters, setting, problem, solution; Sequencing with transition words (first, next, then, finally); Comparing across two texts |
| **Writing** | Narrative Writing (Beginning) | Story planning: character, setting, problem; Writing strong openings (hooks); Dialogue introduction — punctuation and purpose; Connection to enrichment: write a story set in a museum/building |
| **Math** | Multiplication & Division Concepts | Division as sharing and grouping (concrete with counters); Relationship between multiplication and division; Fact families; Word problem schemas: equal groups, comparison |
| **Enrichment Theme** | "Building & Construction" | Mon: Build a nature shelter (outdoor engineering); Tue: Architecture — how buildings tell stories (cultural); Wed: Bridge-building STEM challenge (load testing); Thu: Budget a building project (life skills math); Fri: Collaborative mural — building a community |
| **Assessment** | Weekly Probe #2 + Mid-Cycle Check #1 | ORF, r-controlled vowel sort, multiplication/division probe; Adjust groupings based on 2-week data |
| **IEP Goals Addressed** | Decoding (r-controlled), comprehension (story elements), written expression (narrative), math computation (multiplication/division) |

---

### WEEK 4 — "Breaking It Down"
*Theme: Taking Things Apart to Understand Them*

| Block | Focus | Details |
|-------|-------|---------|
| **Reading/Phonics** | Syllable Division + Multisyllabic Words | VCCV division: rab-bit, nap-kin, bas-ket; VCV division: o-pen, riv-er, mo-ment; Compound words: breaking apart and combining; Consonant-le syllable: ta-ble, sim-ple, puz-zle; Strategy: "chunk it" approach to big words |
| **Comprehension** | Inference & Text Evidence | "I read… I know… So I think…" framework; Finding evidence to support answers; Asking and answering questions about text; Informational text features (headings, captions, diagrams) |
| **Writing** | Narrative Writing (Continued) + Informational | Narrative: writing the middle and end; Adding descriptive details (show, don't tell); Introduction to informational writing: "How Things Work" paragraph; Connection to enrichment: explain how something is built/made |
| **Math** | Fractions (Part 1) | Unit fractions on number lines (concrete: fraction strips); Naming and representing fractions (area models); Equivalent fractions with manipulatives; Comparing fractions: same numerator, same denominator |
| **Enrichment Theme** | "Taking Things Apart" | Mon: Dissecting plants — parts and functions; Tue: How museums preserve artifacts (conservation); Wed: Take apart electronics (safe disassembly + STEM); Thu: Deconstructing a recipe — what does each ingredient do?; Fri: Collage art — deconstructing and reconstructing images |
| **Assessment** | Weekly Probe #3 + Mid-Cycle Check #2 | ORF, multisyllabic word reading list, fraction identification probe; Adjust pacing for final 2 weeks |
| **IEP Goals Addressed** | Decoding (multisyllabic), comprehension (inference), written expression (descriptive detail), math concepts (fractions) |

---

### WEEK 5 — "Real-World Applications"
*Theme: Using What We Know in the Real World*

| Block | Focus | Details |
|-------|-------|---------|
| **Reading/Phonics** | Morphology — Prefixes & Suffixes | Inflectional endings: -s, -es, -ed, -ing, -er, -est; Spelling rules: doubling, drop-e, change-y; Common prefixes: un-, re-, pre-, dis-; Common suffixes: -ful, -less, -ness, -ly; Word webs: base word → derived forms |
| **Comprehension** | Author's Purpose & Text Features | Persuade, inform, entertain; Using text features to locate information; Summarizing informational text; Connecting reading to real-world contexts |
| **Writing** | Opinion/Persuasive Writing | Stating a claim with reasons; Using transition words (first, also, because, finally); Persuasive letter: convince someone to visit a place or try something; Portfolio piece: "Why ___ Matters" essay |
| **Math** | Fractions (Part 2) + Measurement | Comparing fractions with benchmarks (0, ½, 1); Adding/subtracting fractions with like denominators (4th grade); Time, money, and measurement in real-world contexts; Data collection, bar graphs, line plots |
| **Enrichment Theme** | "Real-World Problem Solvers" | Mon: Environmental stewardship — measuring a garden plot; Tue: Community helpers — interview + research project; Wed: Design a solution to a real problem (STEM); Thu: Grocery store math — budgeting a meal; Fri: Persuasive poster/presentation creation |
| **Assessment** | Weekly Probe #4 | ORF, morphology assessment (prefix/suffix meanings), fraction comparison probe, writing rubric (opinion piece) |
| **IEP Goals Addressed** | Decoding (morphology), comprehension (author's purpose), written expression (opinion), math concepts (fractions, measurement) |

### WEEK 6 — "Showcase & Celebrate"
*Theme: Showing What We've Learned*

| Block | Focus | Details |
|-------|-------|---------|

| **Comprehension** | Synthesis & Reflection | Connecting ideas across texts; "How has my thinking changed?"; Reading response: favorite book/text from the summer; Metacognitive reflection: "What reading strategies help me most?" |

| Week | Math Focus | Concrete (Manipulatives) | Representational (Models) | Abstract (Symbols) |

## IEP Goal Alignment Bank

Common 3rd–4th grade IEP goals this program addresses:

### Reading — Decoding
- "Given a list of grade-level words containing [vowel teams / r-controlled vowels / multisyllabic words], [student] will decode with ≥80% accuracy as measured by teacher-created assessments."
- "When presented with multisyllabic words, [student] will apply syllable division strategies to decode with ≥75% accuracy across 3 consecutive probes."

### Reading — Fluency
- "[Student] will read a grade-level passage at ≥[X] words correct per minute with ≥95% accuracy as measured by weekly ORF probes."
- "[Student] will read aloud with appropriate prosody (expression, phrasing, pace) scoring ≥3/4 on the NAEP fluency rubric."

### Reading — Comprehension
- "After reading a grade-level passage, [student] will identify the main idea and 2 supporting details with ≥80% accuracy."
- "[Student] will make inferences using text evidence, answering inferential questions with ≥75% accuracy across fiction and informational text."
- "Given a narrative text, [student] will identify story elements (character, setting, problem, solution) with ≥80% accuracy."

### Written Expression
- "[Student] will write a [narrative/opinion/informational] paragraph with a topic sentence, 3+ supporting details, and a closing sentence, scoring ≥3/4 on a grade-level rubric."
- "[Student] will apply grade-level spelling patterns in written work with ≥80% accuracy as measured by weekly writing samples."

### Math — Computation
- "[Student] will solve [multiplication/division] problems within 100 with ≥85% accuracy as measured by weekly computation probes."
- "[Student] will identify, compare, and order fractions using models and number lines with ≥80% accuracy."

### Math — Problem Solving
- "[Student] will solve 1- and 2-step word problems involving [operations] with ≥75% accuracy, showing work using a model or diagram."

### Executive Function (Cross-Curricular)
- "[Student] will independently use a self-monitoring checklist to complete multi-step tasks with ≥80% independence across 3 consecutive observations."
- "[Student] will demonstrate on-task behavior for ≥[X] minutes during independent work as measured by interval recording."

## Data Collection Framework

### Weekly Progress Monitoring
| Domain | Tool | Frequency | Data Point |
|--------|------|-----------|------------|
| Oral Reading Fluency | 1-minute timed passage (grade-level) | Weekly (Friday) | WCPM + accuracy % |
| Phonics/Decoding | Pattern-specific word list (10–20 words) | Weekly (Friday) | % correct by pattern |
| Comprehension | Maze passage OR 5 comprehension questions | Weekly (Friday) | % correct |
| Writing | Rubric-scored writing sample | Bi-weekly | Rubric score (1–4) per trait |
| Math Computation | Timed probe (20 problems, 2 min) | Weekly (Friday) | # correct, # attempted |
| Math Concepts | 5-question concept check | Weekly (Friday) | % correct |

### Parent Report Data
Each weekly parent update will include:
1. Skills taught this week (brief summary)
2. Progress monitoring scores with trend line
3. Strengths observed
4. Areas for continued practice
5. Home practice suggestion (15 min/night)
6. Preview of next week

---

*This scope and sequence is designed to be responsive. Entry points, pacing, and groupings will be adjusted based on pre-assessment data and ongoing progress monitoring. The Orton-Gillingham framework ensures that instruction is diagnostic and prescriptive — we teach to the student, not just the curriculum.*
