# IEP & Thrive Design System (iOS Pivot)

## Colors
- **Forest (Primary):** #1B4332 (Authority, Trust)
- **Forest Mid:** #2D6A4F (Hovver states)
- **Forest Light:** #40916C (Accents)
- **Sage:** #B7E4C7 (Card accents)
- **Sage Pale:** #D8F3DC (Backgrounds)
- **Cream (Background):** #FDFAF4 (Main background)
- **Cream Deep:** #F5EFE0 (Section backgrounds)
- **Amber:** #D4860B (Highlights, urgency)
- **Amber Light:** #FFF3CD
- **Text:** #1C1917
- **Text Muted:** #78716C
- **White:** #FFFFFF

## Typography
- **Display Font:** Playfair Display (Headings, Hero)
- **Body Font:** DM Sans (UI, Labels, Buttons)
- **Scale:**
  - H1: 3.4rem / Bold / -0.03em letter spacing
  - H2: 2.6rem / Bold / -0.025em letter spacing
  - H3: 1.5rem / Semi-bold
  - Body: 15px / 1.6 line-height

## Style
- **Border Radius:**
  - Pills/Buttons: 100px
  - Cards: 20px
  - UI Elements: 12px
- **Shadows:**
  - Card Hover: 0 8px 32px rgba(27, 67, 50, 0.10)
- **Visual Vibe:** "The Island of Discovery". Lush, natural, calming, and high-quality. Like a premium iPad game.

## Platform
- **Target:** iOS / iPadOS
- **Accessibility:** High contrast support, VoiceOver labels, Dynamic Type compatibility.
