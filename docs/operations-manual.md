**Program:** SPED Summer Intensive

## 1. Program Overview

IEP & Thrive is a 6-week private summer intensive for students with IEPs and learning differences on Long Island, NY. Three program tracks are offered — Reading & Language, Full Academic, and Math & Numeracy — all IEP-aligned, small-group (max 6), and led by credentialed SPED interventionists.

## 3. Enrollment Process

### Pre-Enrollment
1. Parent visits website → submits enrollment inquiry form
2. Operator receives email notification via Resend
3. Operator responds within 24 hours with discovery call link (Calendly)

### Discovery Call (20 min, free)
1. Discuss child's current IEP/504/learning profile
2. Identify primary intervention needs
3. Recommend program track (Reading, Full Academic, or Math)
4. Discuss logistics, pricing, expectations
5. If good fit → send deposit link via Stripe

### Enrollment
1. Parent completes deposit via Stripe ($875 or $1,000)
2. Operator sends enrollment confirmation email
3. Operator sends Parent Intake Form (IEP upload, emergency contacts, medical info, learning preferences)
4. Remaining balance due by June 23, 2026
5. Cohort assignment confirmed by June 30

### Pre-Program IEP Review (Weeks before program)
1. Parent uploads current IEP / 504 Plan / evaluation reports
2. Founder reviews IEP goals, accommodations, and present levels
3. Session plan is mapped to specific annual IEP goals
4. Parent receives "Program Alignment Summary" — a 1-page doc showing how sessions connect to IEP goals

## 4. Progress Reporting

### Weekly Progress Reports (every Friday, emailed as PDF)

**Report Contents:**
- Student name, cohort, program track, week number
- Attendance summary
- Skills targeted this week (mapped to IEP goals)
- Performance data: accuracy percentages, fluency metrics, error analysis
- Behavioral/engagement observations
- Strategies used and student response
- Recommendations for home practice (15 min/night)
- Next week's focus areas

### End-of-Program Comprehensive Report (delivered at program end)

**Report Contents:**
- Student profile and program summary
- IEP goals addressed (with direct alignment citation)
- Pre/post assessment data for each skill area targeted
- Detailed progress narrative with specific examples
- Data visualizations (growth charts, accuracy trends)
- Recommendations for continued support
- Suggested IEP goal language for the September CSE meeting
- This report is designed to be **CSE-ready** — parents can hand it directly to their child's CSE team

## 7. Safety & Emergency Protocols

### Required Documentation (per student)
- Emergency contact form (2 contacts minimum)
- Medical information and allergy disclosure
- Medication authorization (if applicable)
- Photo/video consent form
- IEP / 504 Plan copy
- Signed enrollment agreement

### Emergency Procedures
- **Medical emergency:** Call 911 → administer first aid → notify parent → document incident
- **Fire/evacuation:** Follow posted evacuation route → headcount at assembly point → notify parents
- **Severe weather:** Move to interior room → monitor alerts → hold students until safe pickup
- **Behavioral crisis:** De-escalation protocol → isolate if necessary → contact parent → document incident
- **Unauthorized pickup:** Verify against approved pickup list → do not release → contact parent

### Mandated Reporter Obligations
All staff are mandated reporters under NYS law. Any suspected child abuse or neglect must be reported to the NYS Statewide Central Register (SCR) at 1-800-342-3720.
