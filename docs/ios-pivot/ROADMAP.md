# IEP & Thrive: iOS Pivot Roadmap (Student-First)

**Vision:** Build the definitive, story-driven iOS application for IEP student development. Scalable, gamified, and designed for an Apple Design Award.
**North Star Metric:** Student Session Completion Rate (Engagement) & Free-to-Paid Conversion (Monetization).

---

## Phase 1: Foundation & Story Engine (Weeks 1-4)
*Focus: Locking in the gamified architecture and the "feel" of the app.*

1. **Strategic Documentation**
   * PRD, ADRs, and RFCs finalized.
   * Go/No-Go on iOS Native Architecture.
2. **The "Vibe" & Design Language**
   * Define the narrative universe (Theme, characters, aesthetic).
   * Prototype the Core Haptics and SpriteKit interactions (Make a button click feel incredible).
3. **Core Tech Stack**
   * Xcode project setup with SwiftData and StoreKit 2.
   * CI/CD via Fastlane to TestFlight.

---

## Phase 2: The Learning Prototypes (Weeks 5-8)
*Focus: Translating the curriculum into digital play.*

1. **Literacy MVP (Orton-Gillingham)**
   * Build the "Tactile Sand Tray" for letter tracing.
   * Build the "Speech-to-Text" reading fluency checker.
2. **Math MVP (CRA)**
   * Build the snap-cube physics engine for visual counting.
3. **The Progress Map**
   * Build the scrolling SwiftUI path (the "Journey") that connects the lessons.

---

## Phase 3: Content Scaling & Story Integration (Weeks 9-14)
*Focus: Fleshing out the 6-week curriculum into a continuous app experience.*

1. **Level Generation**
   * Map the 6-week web curriculum into 60+ micro-levels in the app.
   * Implement difficulty scaling algorithms based on student success/failure.
2. **Rewards System**
   * Implement unlockable visual elements, characters, or sandbox areas to keep children engaged.
3. **Accessibility Hardening**
   * Comprehensive VoiceOver audits, high-contrast modes, and reduced motion options.

---

## Phase 4: The Parent Gate & Monetization (Weeks 15-18)
*Focus: Packaging the experience for the buyer.*

1. **Onboarding & Profiling**
   * Quick parent intake flow to set the child's baseline difficulty.
2. **StoreKit Integration**
   * Paywall implementation (e.g., 7-day free trial, then $9.99/mo).
3. **Automated Reporting**
   * Push notification engine for weekly parent updates.
4. **App Store Optimization & Launch**
   * Promotional videos, screenshots highlighting the HIG design.
   * Submission and pitch to Apple App Store editorial team.
